进行laravel的数据迁移的时候，对修改字段类型，需要安装 composer install doctrine/dbal,但是遇到以下问题：

解决如下：
composer install doctrine/dbal --ignore-platform-reqs

如果更新，则：
composer update --ignore-platform-reqs

navicat破解方法

发表于 2017-07-17 |

安装后，在/home/username/.navicat文件夹下有一个system.reg/user.reg文件,删除后即可.
把此文件删除后，下次启动navicat 会重新生成此文件，30天试用期会按新的时间开始计算。
